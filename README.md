# 📚 LibraryHub (CampusVault)

A centralized platform for accessing verified academic notes and PYQs.

---

## 🚀 Live Website

👉 [Click Here](https://campusvault.xyz/)

---

## 🚀 Overview

LibraryHub provides a **structured, moderated, and easily accessible repository** of study materials.

---

## 🧩 Problem

Students rely on:

* WhatsApp groups
* Random Drive links
* Seniors

Result:

* Unorganized data
* Lost resources
* No quality control

---

## 💡 Solution

* Centralized platform
* Verified content (moderation-based)
* Clean categorization
* No login required

---

## ⚙️ System Design

* 📥 Submission → Telegram Bot
* 🛡️ Moderation → Admin
* 🗄️ Storage → Database
* 🌐 Delivery → Website

---

## ✨ Features

* 📂 Notes & PYQs by subject/semester
* ⚡ Fast access
* 🔄 Real-time updates after approval
* 🛡️ Verified content only
